# FB Page Follower Checker

Chrome Extension kiểm tra số Followers của nhiều Facebook Page cùng lúc.

## Tính năng

- Tự động lấy Access Token từ session Facebook đang đăng nhập
- Fetch hàng loạt qua Batch API, xử lý nhanh
- Hiển thị trong Side Panel — không mất dữ liệu khi chuyển tab
- Sắp xếp theo Followers (click header), giữ thứ tự gốc mặc định
- Xuất CSV chuẩn UTF-8

## Cài đặt

1. Mở Chrome → `chrome://extensions/`
2. Bật **Developer mode**
3. Bấm **Load unpacked** → chọn thư mục này
4. Click icon extension → Side Panel mở ra
